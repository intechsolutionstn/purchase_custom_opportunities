Website Cash on Deliver | COD | Pay at Doorstep

This Apps allow customer to order from the website using the Cash on Delivery option. You can pay amount at doorstep, Shop online with COD, COD Payment Method.

Now, manage cash on delivery in Odoo without any hassle. Each transaction is noted in the Odoo with the reference to the Sales order

Features

Easy to manage delivery using the Cash On Delivery method
Administrator can set the minimum order value for the Cash on Delivery order
Administrator can set the maximum order value for the Cash on Delivery order
Configuration to allow delivery options using States and ZIP codes
Payment collection at doorstep
Options to disable some day to exclude the COD option
